# hyderabad.rent

Community-driven, crowdsourced rent transparency map for Hyderabad.

## Tech Stack
- React 18 + TypeScript + Vite
- Mapbox GL JS (map)
- Zustand (state)
- Vitest (tests)
- CSS Modules

## Commands
- `npm run dev` — start dev server
- `npm run build` — production build
- `npx vitest run` — run tests
- `npx tsc --noEmit` — type check

## Env
Copy `.env.example` to `.env.local` and add your Mapbox public token (`pk.***`).
Get one free at https://account.mapbox.com/
